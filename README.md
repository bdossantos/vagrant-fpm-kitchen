# vagrant-fpm-kitchen

Build deb/rpm packages using [fpm][1], [fpm-cookery][2] and [Vagrant][3].

Inspired by [prognostikos][4]'s [project][5].

## Requirements

* Vagrant, yarly
* Debian Vagrant box

## Usage

```
# BUILD ALL THE THINGS
vagrant up

# Build single recipe
RECIPE=recipes/golang/1.4.rb vagrant up
```
